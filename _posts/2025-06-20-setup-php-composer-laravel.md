# Install PHP

Latest 8.4 version:
```bash
/bin/bash -c "$(curl -fsSL https://php.new/install/linux/8.4)"
```

Old composer for Laravel 11:
```bash
/bin/bash -c "$(curl -fsSL https://php.new/install/linux/8.2)"
```

All necessary things are installed:
- PHP
- Composer
- Laravel
- cacert.pem
